## Icon Library (REQUIRED)
Use **Heroicons** for all icons in the game. Heroicons provides free, open-source SVG icons that match the modern, clean aesthetic.

**Setup:**
1. Visit https://heroicons.com/
2. Use the "Outline" style icons (20x20 size recommended)
3. Copy the SVG code directly into your HTML
4. Icons use `currentColor` so they inherit the text color from their parent

**Example Icon Usage:**
```html
<button class="icon-button">
    <svg width="20" height="20" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
        <path d="M21 21l-6-6m2-5a7 7 0 11-14 0 7 7 0 0114 0z"/>
    </svg>
</button>
```

**Common Icons to Include:**
- Search icon (magnifying glass)
- Reset/Refresh icon (arrow path)
- Back/Arrow left icon
- Menu/More options icon (three dots)
- Star icon
- Send/Arrow right icon
- Any game-specific icons needed

**Icon Styling:**
- Icons should use `stroke="currentColor"` or `fill="currentColor"` to inherit parent color
- Standard size: 20x20px for buttons, 16x16px for smaller elements
- Use `stroke-width="2"` for outline icons, `stroke-width="1.5"` for finer details
